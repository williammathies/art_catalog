Export / Import

Export JSON

Download your collection as JSON, respecting current sidebar filters. Paste it into a Claude conversation to get recommendations, generate descriptions, or analyze your collection. Claude can return updated JSON you can import back.

Export HTML Catalog

Download a self-contained HTML catalog with all fields and embedded photos, respecting current sidebar filters. Open in any browser or print for a physical catalog.

Import from JSON

Import a JSON file — either a previous export from this app, or a Claude-generated JSON following the same format. Pieces with matching IDs will be updated; pieces without IDs or with new IDs will be added. Missing fields default to blank. After import you will be taken to the collection.
